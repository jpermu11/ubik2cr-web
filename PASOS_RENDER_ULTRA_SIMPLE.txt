PASO 1: CREAR CUENTA
───────────────────────────────────────────────────────────────
1. Abre: https://render.com
2. Clic en "Get Started for Free"
3. Clic en "Sign up with GitHub"
4. Autoriza Render
5. ✅ Listo

───────────────────────────────────────────────────────────────

PASO 2: CREAR BASE DE DATOS
───────────────────────────────────────────────────────────────
1. Clic en "New +" (arriba derecha)
2. Clic en "PostgreSQL"
3. Llena:
   - Name: ubik2cr-db-oregon
   - Region: Oregon
   - Plan: Free
4. Clic en "Create Database"
5. ESPERA 2 minutos
6. Cuando esté listo, entra a la base de datos
7. Ve a pestaña "Connections"
8. Copia la "Internal Database URL"
   (Algo como: postgresql://...)
9. ✅ GUARDA ESA URL

───────────────────────────────────────────────────────────────

PASO 3: CREAR WEB SERVICE
───────────────────────────────────────────────────────────────
1. Clic en "New +" otra vez
2. Clic en "Web Service"
3. Conecta tu repositorio:
   - Busca: jpermu11/ubik2cr-web
   - Si no aparece, clic en "Configure account"
   - Autoriza GitHub
   - Selecciona: jpermu11/ubik2cr-web
4. Llena:
   - Name: ubik2cr-web
   - Region: Oregon
   - Branch: main
   - Build Command: pip install -r requirements.txt && flask db upgrade
   - Start Command: gunicorn main:app
   - Plan: Free

───────────────────────────────────────────────────────────────

PASO 5: CREAR Y ESPERAR
───────────────────────────────────────────────────────────────
1. Clic en "Create Web Service" (abajo)
