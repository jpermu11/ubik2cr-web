# INSTRUCCIONES DE CONFIGURACIÓN
# ================================

# 1. Crear entorno virtual (recomendado):
#    python -m venv venv
#    venv\Scripts\activate  (Windows)
#    source venv/bin/activate  (Linux/Mac)

# 2. Instalar dependencias:
#    pip install -r requirements.txt

# 3. Configurar variables de entorno:
#    Crear archivo .env con:
#
#    DATABASE_URL=sqlite:///app.db  (para desarrollo local)
#    SESSION_SECRET=tu-clave-secreta-aqui
#    ADMIN_USER=admin
#    ADMIN_PASS=tu-password
#    SMTP_HOST=smtp.gmail.com
#    SMTP_PORT=587
#    SMTP_USER=tu-email@gmail.com
#    SMTP_PASS=tu-app-password
#    SMTP_FROM=Ubik2CR
#    PORT=5000

# 4. Inicializar base de datos:
#    flask db init
#    flask db migrate -m "Initial migration"
#    flask db upgrade

# 5. Ejecutar la aplicación:
#    python main.py
#    O: flask run

# NOTA: Si no configuras DATABASE_URL, usará SQLite local (app.db)
# Para producción con PostgreSQL, configura DATABASE_URL con tu conexión.
